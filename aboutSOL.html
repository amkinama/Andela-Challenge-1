About Stack Overflow Lite

Stack Overflow Lite is a platform where people can ask questions and answers in coding with the various programming languages. Essentially it is made up of the following features:

  1. Users can create an account and log in.
  2. Users can post questions.
  3. Users can delete the questions they post.
  4. Users can post answers.
  5. Users can view the answers to questions.
  6. Users can accept an answer out of all the answers to his/her question as the preferred answer.
